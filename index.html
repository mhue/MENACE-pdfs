About MENACE

MENACE is a mechanical computer built from matchboxes that can learn to play tic-tac-toe. Each matchbox represents a game position, and colored beads inside represent possible moves. The machine "learns" by adding or removing beads based on game outcomes.

How to Use

  1. Generate PDFs: Use the Python scripts to create printable position sheets
  2. Build Boxes: Create physical matchboxes for each position
  3. Use Interface: Our interactive tool helps you identify which box to use during gameplay
  4. Play & Learn: Let MENACE learn by adjusting beads after each game

Requirements

  • Python 3.x for running the generation scripts
  • pdflatex for generating PDF files
  • Modern web browser for the interactive interface
